# EP001 XIAO ESP32C3 v0.9 Test Procedure

1. Close KidsBlock completely.
2. Run `restore_ep001_xiao_esp32c3_v0_8_safe.bat`.
3. Start KidsBlock and confirm a normal ESP32 sketch can compile again, if needed.
4. Close KidsBlock.
5. Run `install_ep001_xiao_esp32c3_v0_9.bat`.
6. Start KidsBlock.
7. Select Seeed Studio XIAO ESP32C3.
8. Compile and upload a simple blink sketch.
9. Report the exact error if it fails.

To restore normal ESP32 behavior, run `restore_ep001_xiao_esp32c3_v0_9.bat`.
